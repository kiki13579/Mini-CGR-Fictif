À propos de Mini CGR Fictif

1. Notre mission

Mini CGR Fictif est un projet fictif conçu dans un cadre pédagogique. Il a pour but de simuler une plateforme de réservation de films en ligne, avec une interface moderne, interactive et accessible.

2. Créateur du site

Ce site a été entièrement imaginé, développé et designé par Kellyan Bernard dans le cadre d'un projet web. Le site n’a aucun lien avec les cinémas CGR ou toute autre entité commerciale.

3. Objectifs pédagogiques

Le but de ce projet est de mettre en application des compétences en HTML, CSS, JavaScript, UX/UI, et responsive design, tout en respectant les bonnes pratiques du développement web.

4. Technologies utilisées

Le site a été développé en HTML5, CSS3 et JavaScript vanilla. Le code est optimisé pour un affichage responsive sur ordinateurs, tablettes et smartphones.

5. Contact

Pour toute question ou remarque, vous pouvez me contacter via les informations présentes sur ma page de profil ou en consultant les autres sections légales du site.
